# 閃ｰセンｰ (SEN)

表示された一文字を入力するタイピング練習ゲームです。  

推奨環境: Windows + Google Chrome
PC物理キーボードでのプレイを推奨します。  
モバイルや仮想キーボードでは動作が不安定です。

### プレイ方法
→ [ここをクリックしてプレイ](https://tk-laboratory.github.io/SEN/)
（GitHub Pagesで公開中。）

### 主な機能
- 時間制限: 15秒 / 30秒 / 60秒
- 難易度: Easy（アルファベットのみ） / Normal（記号・Shiftキー不使用） / Hard（記号・Shiftキー使用）
- キーボードレイアウト: JIS / US 切り替え
- 一定時間入力がない場合にキーボードガイドにヒント表示
- 結果分析
  - 反応時間グラフ
  - 入力ログ一覧
  - 文字別平均時間・最速/最遅
  - ランク別集計

### 操作
- Enter または Space: 開始
- 表示された文字を入力
- Esc: 中断 / メニューに戻る

結果画面では G / D / S キーまたはボタンで表示を切り替え可能です。  
X キーで結果をTwitterに投稿できます。
